# 🪽 AQTIVE
LXQt on Arch made effortless.
## 📜 System Requirements
| Component | Minimum | Recommended |
| :--- | :--- | :--- |
| **Processor** | 64-bit x86-64 | Dual Core or better |
| **RAM** | 1GB | ≥2GB |
| **Storage** | 4GB | ≥8GB |
| **Architecture** | Arch Linux | Post-Archinstall |

---

## 🚀 Installation
Run this command after using archinstall as a user with `sudo` privileges. 

```bash
curl -fsSL https://is.gd/aqtive | sh
```

---

To update your aqtive setup, re-run the command above ^^
